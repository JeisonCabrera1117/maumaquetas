# Maumaquetas - Web de Maquetas Artesanales

Sitio web moderno y optimizado para mostrar y vender maquetas artesanales, desarrollado con React, TypeScript y Tailwind CSS.

## 🚀 Tecnologías

- **React 18** con TypeScript
- **Vite** como bundler
- **Tailwind CSS** para estilos
- **Framer Motion** para animaciones
- **React Helmet Async** para SEO

## 📦 Instalación

```bash
npm install
```

## 🛠️ Desarrollo

```bash
npm run dev
```

La aplicación se ejecutará en `http://localhost:3000`

## 🏗️ Build

```bash
npm run build
```

Los archivos generados estarán en la carpeta `dist/`

## 🌐 Despliegue en Vercel

1. Sube el código a un repositorio de GitHub
2. Ve a [Vercel](https://vercel.com)
3. Importa el repositorio
4. Vercel detectará automáticamente la configuración de Vite
5. ¡Despliega!

## 🎨 Secciones

- **Inicio**: Hero section con presentación
- **Sobre mí**: Información personal y materiales utilizados
- **Galería**: Grid de proyectos por categorías (Edificios, Iglesias, Casas, Vehículos)
- **Servicios**: Tipos de maquetas que se crean
- **Contacto**: Botón directo a WhatsApp

## 🔎 SEO

- Meta tags optimizados
- Open Graph para redes sociales
